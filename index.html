Курс “Куратор”

Системна програма для спеціалістів, які хочуть працювати структурно, доказово та впевнено.

Для кого курс

  • для нутріціологів, які хочуть більше структури
  • для лікарів та фахівців у сфері здоров’я
  • для спеціалістів, які хочуть підвищити цінність консультацій

Програма курсу

01. Клінічне мислення та алгоритм роботи

Навчитеся працювати системно, а не інтуїтивно.

  • як правильно збирати анамнез
  • як аналізувати запит клієнта та знаходити справжню причину проблеми
  • червоні прапори: коли клієнта потрібно направити до лікаря
  • алгоритм першої консультації
  • логіка ведення клієнта від першої зустрічі до результату

02. Доказові інструменти та суплементація

Без хаотичних призначень і модних схем.

  • які аналізи справді важливі у роботі спеціаліста
  • інсулінорезистентність та метаболічний синдром
  • робота з дефіцитами
  • основи доказової суплементації
  • як аргументувати свої рекомендації клієнту

03. Супровід клієнта та розвиток практики

Перетворіть знання на результат для клієнта та стабільний дохід для себе.

  • як побудувати консультацію, щоб клієнт бачив цінність вашої роботи
  • робота зі зривами та низькою мотивацією
  • комплаєнс: як допомогти клієнту дотримуватись рекомендацій
  • довгостроковий супровід
  • позиціонування експерта та підвищення вартості послуг

Що ви отримаєте

  • чітку систему роботи з клієнтом
  • алгоритми консультацій
  • впевненість у рекомендаціях
  • більше експертності та структури
